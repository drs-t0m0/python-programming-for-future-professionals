# まとめ

## FizzBuzzプログラム

- 3で割り切れる数値を引数に渡すと、“Fizz”を返す。
- 5で割り切れる数値を引数に渡すと、“Buzz”を返す。
- 15で割り切れる数値を引数に渡すと、“Fizz Buzz”を返す。
- それ以外の数値はその数値を文字列に変えて返す。
